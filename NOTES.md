  + tools (installations and introductions):
    + brew / chocolatey, package managers at a high level
    + terminal / git bash
    + text editor
    + conda
    + python
    + pip
    + git
  + command-line computing:
    + terminal / git bash commands (pwd, cd, ls, mkdir, mv, rm, touch, cat, etc.)
